The idea

An X-ray for what the
model is thinking.

LLM X-Ray makes the inside of a language model legible — attention, token predictions, and the flow of data through layers, rendered in real time.

The identity is built on one move: a specimen placed under a scanner. The X of “X-Ray” sits inside registration crop-marks, with a single focal node where the instrument is looking. Clinical precision, warmed by paper and a single living accent.

03 · The mark

Anatomy

Three parts, one idea. Every element is geometric and reproducible from the construction grid.

A
Registration crop-marks
Four corner brackets frame the subject — the language of a scan bed and a viewfinder. Set at 34% opacity so they never compete with the X.
B
The X
Two rounded strokes at exact 45° diagonals. Carries the “X-Ray” name and reads cleanly down to favicon scale.
C
Focal node
A single accent dot at dead center — the point the instrument is looking at. The only place color is allowed inside the mark.

08 · Spacing

Clear space & minimum size

Clear space

Keep a margin of X on all sides, where X is the height of the crop-mark bracket. Nothing — type, edges, other logos — enters this zone.
