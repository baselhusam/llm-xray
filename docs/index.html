Run it locally

The live app needs a running backend (Qwen3-1.7B on CPU) and frontend. The fastest path is Docker: 

git clone https://github.com/baselhusam/llmxray
cd llmxray
make up          # build + start backend + frontend
# open http://localhost:3000

First boot downloads the Qwen3-1.7B weights once (~7 GB, cached afterwards). See the repo README for the two-terminal dev setup without Docker.
